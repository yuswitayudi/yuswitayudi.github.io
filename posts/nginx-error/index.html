nginx: [emerg] bind() to 0.0.0.0:80 failed (98: Address already in use)

Oct 28, 2022 17:02 · 32 words · 1 minute read

Untuk mengatasi error nginx seperti dibawah ini

nginx: [emerg] bind() to 0.0.0.0:80 failed (98: Address already in use)

Jalankan perintah berikut

sudo fuser -k 80/tcp

service nginx start

Semoga aman servermu :))
